
ขั้นตอนการนำไฟล์เข้า git
1.    คลิกขวาในโฟลเดอร์(ที่หน้าแรก) เลือก open in terminal แล้วพิมพ์
2.  git add .
3.  git commit -m “ <พิมพ์ประวัติที่เปลียนแปลง(พิมพ์อะไรก็ได้)> ”
4.  git push


ดาวน์โหลดโปรเจคเข้าเครื่อง : git clone https://github.com/EverytimeAomsin/HIGHWAY18-build-road.git
อัพเดทไฟล์ที่เปลียนแปลงใน Git มาอัพเดทไฟล์ในโฟลเดอร์นี้ : git pull
